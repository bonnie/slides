# Hydration

<v-clicks>

- Server renders React component 
  - _without_ interactivity
  - no event handlers, state, etc
  - can't respond to user input
- Client receives
  - rendered component
  - JavaScript
- Client _hydrates_ component
  - adds JavaScript for interactivity

</v-clicks>
